# RNG

硬件随机数

随机数发生器用于生成随机数的程序或者硬件, STM32的RNG处理器是一个以连续噪声为基础的随机数发生器, 提供一个32位的随机数

有的型号使用的是真随机数发生器, 而伪随机数使用确定的算法计算出来随机数, 不是真的随机数

应用场景, 验证码, 密码学, 概率学, 统计学, 游戏
