# USMART

一个串口的调试组件, 可以用来提高代码的调试效率

可以调用大部分用户直接编写的函数, 按照格式放在注册表里面, 并且占用的资源比较少, 需要4KB的FLASH以及72B的SRAM

支持的参数种类比较多, 整形, 字符串, 函数指针

支持函数返回值的显示以及对格式进行显示

还可以计算函数执行的时间

通过用户输入的字符串和本地函数名, 通过函数指针调用不同的函数
